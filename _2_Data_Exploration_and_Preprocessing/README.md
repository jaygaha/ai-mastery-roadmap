# Data Exploration and Preprocessing

This module focuses on the critical initial steps of any AI/ML project: understanding your data. Before building models, you must explore, clean, and prepare your data to ensure valid results.

## Modules

### 1. Understanding Data Types, Sources, and Loading Techniques
Located in: `_01_Understanding_Data_Types_Sources_and_Loading_Techniques`

Learn the foundational skills of:
- Identifying different data types (Numerical, Categorical, Text, etc.).
- Recognizing common data sources (Databases, CSV, JSON, etc.).
- Loading data efficiently using Pandas.
